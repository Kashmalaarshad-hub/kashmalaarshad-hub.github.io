## The Syntax War Begins ⚔️

My first proper week of 2nd semester and already my brain feels like it's running out of RAM!

## What I Learned This Week

### OOP in C# 

We started Object Oriented Programming in C#. Coming from Python in 1st semester, C# feels very *strict* — every variable needs a type, every class needs proper structure. But slowly I'm getting it!

I made my first `BankAccount` class with deposit and withdrawal methods. Seeing it work was honestly so satisfying!

```csharp
class BankAccount {
    public string Owner;
    public double Balance;
    
    public void Deposit(double amount) {
        Balance += amount;
    }
}
```

### Digital Logic Design 🔌

DLD is fascinating! We're learning about logic gates — AND, OR, NOT, NAND, NOR. It's like the foundation of how computers actually work at the hardware level.

Boolean algebra is tricky but once you understand K-maps, simplification becomes fun (kind of)!
